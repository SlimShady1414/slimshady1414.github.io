This project utilizes Streamlit, Hugging Face Transformers (LaMini-T5-738M, LLAMA), LangChain (CSV Loader, Chroma), Sentence Transformer (all-MiniLM-L6-v2), FAISS, MiniLM, and Tempfile to create interactive chatbots for processing and analyzing PDF and CSV data.
