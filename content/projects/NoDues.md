This project is a dues management system designed for students. It allows students to authenticate using their Student Registration Number (SRN) and password, view their dues for various departments, and clear them individually. If all dues are cleared, the system generates a certificate for the student. The program ensures secure authentication through passwords stored in a file and maintains dues information in another file.
